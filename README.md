# I-consolePro 落地页

一个现代化的应用下载落地页，为 I-consolePro 智能健身应用提供多平台下载入口。

## 📋 项目简介

I-consolePro 落地页是一个基于 Vue 3 构建的响应式单页应用，为用户提供便捷的应用下载入口。页面支持中英文双语切换，适配移动端、平板和桌面端多种设备。

## ✨ 主要特性

- 🌍 **多语言支持**：支持中文和英文切换，语言偏好自动保存
- 📱 **响应式设计**：完美适配移动端、平板和桌面端
- 🎨 **现代化 UI**：渐变色彩、流畅动画、优雅的交互体验
- 🔗 **多平台下载**：支持 Google Play、iOS、本地下载等多种渠道
- 💡 **智能引导**：移动端提供下载引导提示，提升用户体验
- ⚡ **高性能**：基于 Vite 构建，快速开发和构建

## 🛠️ 技术栈

- **框架**：Vue 3 (Composition API)
- **构建工具**：Vite 5
- **开发语言**：JavaScript
- **样式**：CSS3 (Scoped Styles)

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0 或 yarn >= 1.22.0

### 安装依赖

```bash
npm install
```

### 开发模式

启动开发服务器（默认端口 3000）：

```bash
npm run dev
```

开发服务器将在 `http://localhost:3000` 启动，并自动在浏览器中打开。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产构建

```bash
npm run preview
```

## 📱 功能说明

### 多语言切换

- 点击导航栏右上角的语言切换按钮，可在中文和英文之间切换
- 语言偏好会自动保存到浏览器的 localStorage 中
- 下次访问时会自动加载上次选择的语言

### 下载渠道

页面提供以下下载渠道：

- **Google Play**：跳转到 Google Play 商店
- **iOS 下载**：iOS 设备下载入口（移动端会显示提示弹窗）
- **本地下载**：直接下载 APK 文件
- **应用宝、小米、华为、三星**：国内应用商店（当前已隐藏，可通过修改组件中的 `v-if` 属性启用）

### 移动端引导提示

在移动端或平板设备上点击下载按钮时，会显示引导提示条，帮助用户了解如何在浏览器中打开下载链接。

### iOS 下载提示

在移动端或平板设备上点击 iOS 下载按钮时，会显示"暂未开放"的提示弹窗。

## 🎨 样式特性

- **渐变背景**：页面采用蓝绿色渐变背景
- **渐变文字**：Logo 和下载渠道文字使用渐变色效果
- **毛玻璃效果**：导航栏使用 backdrop-filter 实现毛玻璃效果
- **流畅动画**：按钮悬停、提示条显示等都有平滑的过渡动画
- **响应式布局**：使用 CSS Grid 和 Flexbox 实现灵活的布局

## 🔧 配置说明

### 修改端口

在 `vite.config.js` 中修改 `server.port` 配置：

```javascript
server: {
  port: 3000,  // 修改为你想要的端口
  open: true
}
```

### 添加新的下载渠道

1. 在 `src/components/LandingPage.vue` 中添加新的渠道项
2. 在 `src/utils/i18n.js` 中添加对应的翻译文本
3. 将渠道图标添加到 `src/image/` 目录

### 自定义样式

主要样式定义在 `src/components/LandingPage.vue` 的 `<style scoped>` 部分，可以根据需要修改颜色、字体、间距等。
